# GO Mobility — Complete Admin Dashboard

## Quick Start
```
npm install
npm run dev
```
Open: http://localhost:5173

## Admin Panel Features (Both Roles)
- Dashboard (live stats, charts, heatmap)
- User Management (profile modal, wallet, block)
- Driver Onboarding (KYC, documents)
- Ride Monitoring (force cancel, SOS, live)
- Finance (transactions, failed payments, platform fee log)
- Refunds, Driver Payouts
- Pricing Engine (fares + surge history)
- Subscriptions (plans, revenue chart, expired users)
- City Management (India map, zones, geo-fence, heatmap)
- Dispatch Settings, Driver Incentives
- Reviews (flagged + rating trend analytics)
- Complaints & Support (SLA tracking)
- Operational Metrics, Push Notifications, Settings
